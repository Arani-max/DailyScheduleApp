# Распорядок дня — Windows Forms приложение

## Описание
Приложение для планирования и отслеживания распорядка дня с графическим интерфейсом 
на Windows Forms.

## Функционал

### Управление задачами
- Создание, редактирование и удаление задач
- Указание времени, категории, приоритета и описания
- Отметка задач как выполненных
- Обнаружение пересечений по времени
- Цветовая индикация приоритета и статуса

### Просмотр и фильтрация
- Календарь для навигации по дням
- Фильтр по категории
- Фильтр по приоритету
- Отображение только невыполненных задач
- Подсветка текущей задачи

### Статистика
- Общая статистика за выбранный период
- Статистика по категориям
- Анализ невыполненных задач
- Отображение свободных временных слотов

### Файловые операции
- Автоматическое сохранение в JSON
- Экспорт расписания дня в текстовый файл
- Импорт задач из текстового файла
